Funktionalitet på administrator niveau (club_admin)

  • Påkrævet admin niveau: club_admin
  • Komplet CRUD (For alle brugere):
    • Se alle sider/oversigter
    • Oprette medlemmer (Via admin og csv import), Redigere medlemmer, Slette medlemmer
    • Oprette klub events, Redigere klub events, Slette klub events
    • Oprette både, Redigere både, Slette både
  • Facebook:
    • Indtaste og redigere api informationer til klubbens Facebook side
    • Poste oprettede klub events direkte fra admin til Facebook sidens wall (Billede, beskrivelse, link m.m.)
  • Abbonnements oversigt for klubben
  • Skrive informations beskeder/opslag på klub

Funktionalitet på bruger niveau (club_admin & club_user)

  • Påkrævet admin niveau: club_admin & club_user
  • Komplet CRUD (For egen bruger):
    • Se alle sider/oversigter
    • Redigere egen profil, Slette egen profil
    • Oprette ture, Redigere egne oprettede ture, Slette egne oprettede ture
      • Give turen en status (Åben: Alle kan melde sig til den - Lukket: Egen tur - Delt: Invitere udvalgte medlemmer)
    • Fra- og tilmelde sig til andres ÅBNE ture (hvis der er plads)
    • Oprette grupper (Invitere andre medlemmer)
      • Oprette gruppe events/ture, redigere egne oprettede gruppe events/ture, slette egne oprettede gruppe events/ture
  • Skrive opslag på klub admin wall
  • Facebook:
    • Poste sin(e) sejltur(e) på sin egen Facebook sides wall (Billede, beskrivelse m.m.)
  • Modtage SMS 1 døgn inden en tilmeldt tur/event starter

Funktionalitet på tværs af klubber (club_admin)

  • Påkrævet admin niveau: club_admin
  • CRUD:
    • Flytte/kopiere et medlem fra en anden klub, såfremt den anden klub bruger systemet OG har godkendt flytning
    • Oprette shared klub events, Redigere shared klub events, Slette shared klub events
      • Invitere udvalgte klubber
  • Skrive besked til andre klubber hvis denne mulighed er 'ticked of' i settings
